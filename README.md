At the Bus Stop
=================
An app for viewing information on Studio Ghibli films and leaving personal reviews and ratings on them.

Technologies
------------
* JavaScript
* HTML
* CSS
* Morgan
* Mongoose
* MongoDB
* Express
* React
* NodeJS
* Heroku
* Netlify
* Studio Ghibli API
